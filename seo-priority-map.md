## First Search Console dashboard cuts

When Search Console access is available, create and review these cuts first:

1. Queries for the 10 priority pages above.
2. Pages with impressions but average position worse than 20.
3. Pages with CTR under 2% and impressions above 50.
4. English vs Chinese performance split.
5. Index coverage for `/learn/` and `/zh/learn/`.

## First optimization loop

1. Find the top 20 impression queries in GSC.
2. Map each query to one canonical AILD page.
3. Merge or de-emphasize pages that overlap too heavily.
4. Rewrite titles/meta/H1/intros for the pages with the best early impression signals.
5. Add internal links from support pages into the winning pillars using descriptive anchor text.
